## 3-2
### 回答
```puml
@startuml
class ユーザー {
    メールアドレス
    パスワード
}

class 特典 {
    有効期限
}

class アフィリエイトID指定 {

}

ユーザー "1" -- "*" 特典
特典 <|-- アフィリエイトID指定
@enduml
```

### 正解
```puml
@startuml
class ユーザー {
    メールアドレス
    パスワード
}

class 有料オプション {
    有効期限
    アマノガワアフィリエイトID
}

ユーザー "1" -- "0..1" 有料オプション
@enduml
```
